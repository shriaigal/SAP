Student Academic Portal

A Student Management System (SMS) is a web-based application designed to efficiently manage student data, attendance, marks, and feedback. The system will have three user roles: Admin, Teacher, and Student, each with different levels of access and functionalities. The authentication system will use Flask-Login, ensuring secure user login and role-based access. Admins can add, edit, and delete student records, update and manage attendance, and view reports. Teachers can update student marks, while Students can only view their marks, attendance, and personal details. The attendance module will allow teachers to mark daily attendance and generate reports. The marks management system will allow teachers to enter, edit, and update student exam scores, which will be displayed using Chart for graphical representation. Students can also download their marks in PDF format. A timetable and academic calendar feature will help students keep track of their schedules. Additionally, a feedback system will be integrated where students can submit their feedback, which will be store in a database. The project structure will include separate modules for authentication, student management, attendance, and marks. The database schema will be designed to handle student profiles, attendance logs, and exam scores efficiently.
